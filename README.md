# Bullet_OP3
Use Pybullet to replace Gazebo and Webots simulator. 

## Development Installation
`pip install -e .`

## Start OP3 Walking
<img src="docs/walker_demo.gif" width="80%"/>

* Run `python scripts/start_walking.py`

## Grabbing Ball
* Run `python scripts/op3_arm.py`
